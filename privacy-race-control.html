Mumbles Raft Race Control

Privacy Policy

Overview

Mumbles Raft Race Control is an organiser application used to send official live event notifications during the Mumbles RNLI Raft Race.

The application is intended for use by authorised event organisers and race control personnel.

Personal Information

The application does not require a user account and does not ask users to provide names, addresses, telephone numbers, email addresses, payment information or other personal details.

  • No user registration.
  • No public user profiles.
  • No location collection.
  • No advertising.
  • No cross-app tracking.
  • No sale of personal information.

Notification Messages

When an authorised operator sends a notification, the notification title and message are transmitted securely to a Cloudflare Worker and then to Firebase Cloud Messaging for delivery to users of The Mumbles Raft Race app.

Notification messages may contain official event information such as opening times, race updates, delays, safety information, prize-giving announcements or cancellation notices.

Security

Communication between the application and the notification service is protected using HTTPS.

Firebase service credentials are stored securely on the server and are not stored inside the application.

Third-Party Services

The application uses the following services to provide notification delivery:

  • Cloudflare Workers.
  • Firebase Cloud Messaging.
  • Apple Push Notification Service.

These services may process technical information required to deliver notifications and operate under their own privacy policies and terms.

Analytics and Advertising

The application does not contain advertising and is not used for behavioural advertising or cross-app tracking.

Matthew Moore Apps does not sell personal information.
